Overview

This application shows off Lovefield, a Google product that provides a relational database in JavaScript. This enables easy and fast client-side management of large datasets.

Usage

  • This application uses a dataset of the 5000 largest cities in the United States. To peruse the dataset navigate to Data.
  • To try out Lovefield's query capabilities take a look at Try it Out.
  • Compare provides a comparison of raw JavaScript vs. Lovefield performance in common scenarios.

Implementation

Data is loaded into Lovefield once on initialization. For all comparison operations all JS-only operations perform a service call to acquire data then use custom logic to sort/filter the data as necessary. Lovefield operations acquire data from within Lovefield and use its built-in sort/filter capabilities. While it is true that the service call artificially inflates Javascript performance it more accurately reflects common scenarios since manually storing large amounts of data in-memory in Javascript is generally frowned upon.
